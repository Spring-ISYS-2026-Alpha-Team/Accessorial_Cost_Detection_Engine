# PACE UI Prototypes (Static HTML/CSS)

These files are **implementation-ready visual prototypes** for Google Antigravity to mirror in Streamlit.

## What’s included
- `index.html`: Public Home (Landing)
- `login.html`: Login
- `user-home.html`: Authenticated landing (welcome + stats + animation placeholder)
- `dashboards.html`: Analytics dashboard layout with Chart.js placeholders
- `components.html`: Component library / style reference
- `shared.css`: Design tokens + components + responsive layout
- `shared.js`: Minimal interactions (dropdowns, chart placeholders)

## How to view
Open any HTML file in a browser (double-click).

## Notes
- Charts use the Chart.js CDN and render **placeholder data** only.
- Visual theme matches the requested palette from `design/PACE_UIUX_Design_Spec_Antigravity.md`.
